# Pokemon Combat Dashboard

This project is a Streamlit app that allows users to:

- Select two Pokémon
- Compare their stats
- Visualize their attributes
- Simulate a battle based on their total stats

## Technologies used
- Python
- Streamlit
- PokeAPI
- Pandas
- Plotly

## How to run the app

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
